Introduction

The ATS-SDK includes Python sample programs that demonstrate many of the types of acquisitions possible with AlazarTech digitizer boards. These sample programs are organized into folders, with one folder per board model.

Please refer to the AlazarTech SDK Guide for more information about using these sample programs.
